> ## apicloud 代码文件的命名规范
>
> 文件夹命名，文件命名不能带 - 或多个. 不要有大写字符
> 
> 正确文件命名格式举例：login.js
> 
> 错误文件命名格式举例：login-test.js; login.test.js; login-test-js.js
> 
> 正确文件夹命名格式举例： loginhtml
> 
> 错误文件夹命名格式举例： login-html; login.html
